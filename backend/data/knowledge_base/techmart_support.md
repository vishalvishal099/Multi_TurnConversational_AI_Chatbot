## Troubleshooting Guides

### Laptop Issues

#### Laptop Won't Turn On
1. **Check Power:** Ensure charger is connected and outlet works. Try a different outlet.
2. **Perform Hard Reset:** Hold power button for 15 seconds, release, wait 10 seconds, then press power again.
3. **Check Battery:** If removable, remove battery, hold power 30 seconds, reinstall battery.
4. **Check Display:** Connect to external monitor to rule out display issues.
5. **Check Charging LED:** If no LED when plugged in, charger may be faulty.
6. **Still Not Working:** Contact support for warranty repair.

#### Laptop Running Slow
1. **Restart:** Simple restart can resolve many issues.
2. **Check Storage:** Ensure at least 10% free disk space. Delete unnecessary files.
3. **Close Background Apps:** Use Task Manager (Ctrl+Shift+Esc) to end unnecessary processes.
4. **Check for Malware:** Run antivirus scan.
5. **Update Drivers:** Check Device Manager for outdated drivers.
6. **Check for Updates:** Install latest Windows updates.
7. **Consider RAM Upgrade:** If using 8GB, upgrade to 16GB can help.

#### Laptop Overheating
1. **Clean Vents:** Use compressed air to clean dust from vents.
2. **Use on Hard Surface:** Avoid soft surfaces that block airflow.
3. **Check Fan:** If fan not spinning, contact support.
4. **Reduce Load:** Close resource-heavy applications.
5. **Use Cooling Pad:** Consider purchasing a laptop cooling pad.
6. **Thermal Paste:** For older laptops, thermal paste may need replacement (professional service recommended).

#### Laptop Battery Draining Fast
1. **Check Battery Health:** Settings → System → Power → Battery health
2. **Reduce Brightness:** Lower screen brightness to 50-70%.
3. **Disable Bluetooth/WiFi:** When not in use.
4. **Check Background Apps:** Some apps drain battery in background.
5. **Enable Battery Saver:** Windows → Settings → Battery → Battery saver
6. **Update BIOS:** Sometimes battery issues fixed with BIOS updates.
7. **Battery Replacement:** If over 2 years old, battery may need replacement.

---

### Smartphone Issues

#### Phone Won't Charge
1. **Check Cable:** Try different USB-C cable.
2. **Check Adapter:** Try different power adapter (ensure it supports fast charging if needed).
3. **Clean Port:** Gently clean charging port with toothpick or compressed air.
4. **Restart Phone:** Hold power button and restart.
5. **Try Wireless Charging:** If phone supports it, test wireless charging.
6. **Safe Mode:** Boot to safe mode to rule out app issues.
7. **Factory Reset:** Last resort - backup data first.

#### Phone Screen Unresponsive
1. **Clean Screen:** Wipe screen with microfiber cloth.
2. **Remove Screen Protector:** Temporarily remove to test.
3. **Restart Phone:** Force restart by holding power + volume down for 10 seconds.
4. **Check for Updates:** Install latest software updates.
5. **Safe Mode Test:** If works in safe mode, an app is causing issues.
6. **Factory Reset:** After backing up data.

#### Phone Battery Draining Fast
1. **Check Battery Usage:** Settings → Battery → See which apps use most battery.
2. **Reduce Screen Brightness:** Enable adaptive brightness.
3. **Disable Location Services:** For apps that don't need it.
4. **Turn Off Always-On Display:** If enabled.
5. **Limit Background Apps:** Settings → Apps → Background restrictions.
6. **Check for Rogue Apps:** Uninstall recently installed apps.
7. **Enable Battery Saver Mode:** In Settings → Battery.

#### Phone Running Slow
1. **Restart Phone:** Clears temporary memory.
2. **Clear Cache:** Settings → Storage → Clear cache.
3. **Uninstall Unused Apps:** Free up storage and memory.
4. **Check Storage:** Ensure at least 10% free storage.
5. **Disable Animations:** Developer options → Reduce animation scale.
6. **Factory Reset:** If persistent issues after backup.

---

### Headphones Issues

#### Headphones Won't Connect via Bluetooth
1. **Check Battery:** Ensure headphones are charged (LED indicators).
2. **Enable Pairing Mode:** Hold power button for 5 seconds until LED flashes blue/red.
3. **Forget Device:** On phone/computer, forget the headphones and re-pair.
4. **Reset Headphones:** Hold power + volume down for 10 seconds.
5. **Check Bluetooth Range:** Stay within 30 feet (10 meters) of device.
6. **Update Firmware:** Check TechMart app for firmware updates.
7. **Try Different Device:** Test with another phone or computer.

#### Audio Quality Issues
1. **Check Codec:** Ensure LDAC or aptX is enabled if supported.
2. **Reduce Distance:** Move closer to source device.
3. **Check for Interference:** Move away from WiFi routers, microwaves.
4. **Clean Ear Pads:** Debris can affect sound quality.
5. **Reset EQ Settings:** Return to default equalizer settings.
6. **Update Firmware:** Install latest updates via TechMart app.

#### ANC Not Working Properly
1. **Check Fit:** Ensure proper seal with ear cups/tips.
2. **Check ANC Mode:** Ensure ANC is enabled, not transparency mode.
3. **Clean Microphones:** Gently clean external ANC mics.
4. **Reset ANC:** Toggle ANC off and on.
5. **Update Firmware:** ANC performance often improved with updates.
6. **Environmental Factors:** ANC works best on constant low-frequency sounds.

### Smartwatch Issues

#### Watch Won't Sync
1. **Check Bluetooth:** Ensure Bluetooth enabled on phone.
2. **Check App:** Ensure TechMart Wear app is installed and updated.
3. **Restart Both Devices:** Restart watch and phone.
4. **Re-pair:** Unpair watch from app and phone Bluetooth settings, then re-pair.
5. **Check Permissions:** Ensure app has necessary permissions (location, notifications).
6. **Clear App Cache:** Settings → Apps → TechMart Wear → Clear cache.

#### Watch Battery Draining Fast
1. **Reduce Always-On Display:** Disable or reduce timeout.
2. **Limit Notifications:** Only enable essential app notifications.
3. **Reduce Health Monitoring:** Disable continuous heart rate if not needed.
4. **Disable Raise-to-Wake:** Use button to wake instead.
5. **Update Software:** Battery optimizations in updates.
6. **Check for Stuck Apps:** Restart watch to clear stuck processes.

#### Inaccurate Health Readings
1. **Proper Fit:** Watch should be snug, 1 finger width above wrist bone.
2. **Clean Sensors:** Wipe back of watch with damp cloth.
3. **Skin Contact:** Remove watch, wait 30 min, reattach firmly.
4. **Restart Watch:** Clears sensor calibration issues.
5. **Update Software:** Sensor accuracy improvements in updates.
6. **Manual Calibration:** Some features allow manual calibration in settings.
